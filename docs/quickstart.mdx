<Info>
  **Prerequisites**

  - TopK account ([Sign up here](https://console.topk.io/signup))
  - TopK API key ([Get API key here](https://console.topk.io/api-key))
</Info>

<Tabs>
  <Tab title="CLI" icon="terminal">
    <Steps>
      <Step title="Install CLI">
        ```bash
        brew tap topk-io/topk
        brew install topk
        ```

        <Tip>If you don't have Homebrew installed, you can install it [here](https://brew.sh).</Tip>
      </Step>
      <Step title="Authenticate">
        ```bash
        topk auth
        ```

        This command prompts you to either create a new API key or configure an existing one for the CLI, or you can provide your API key via the `TOPK_API_KEY` environment variable:

        ```bash
        export TOPK_API_KEY="your-api-key"
        ```
      </Step>
      <Step title="Upload a file">
        Upload a file to your dataset:

        ```bash
        topk upload ./report.pdf --dataset quickstart
        ```

        <Info>The `--region` flag determines where your data is stored. See [available regions](/regions).</Info>
      </Step>
      <Step title="Ask a question">
        ```bash
        topk ask "What are the key findings?" -d quickstart
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Python SDK" icon="/icons/python.svg">
    <Steps>
      <Step title="Install Python SDK">
        <CodeGroup>
          ```bash pip
          pip install topk-sdk
          ```
          ```bash uv
          uv add topk-sdk
          ```
        </CodeGroup>
      </Step>
      <Step title="Initialize the client">

        Setup the TopK client with your API key and region.

        <CodeGroup>
          ```python Sync
          import os
          from topk_sdk import Client

          client = Client(
              api_key=os.environ["TOPK_API_KEY"],
              region=os.environ.get("TOPK_REGION", "aws-us-east-1-elastica"),
          )
          ```
          ```python Async
          import os
          from topk_sdk import AsyncClient

          client = AsyncClient(
              api_key=os.environ["TOPK_API_KEY"],
              region=os.environ.get("TOPK_REGION", "aws-us-east-1-elastica"),
          )
          ```
        </CodeGroup>
        <Info>See [available regions](/regions) for a full list of supported regions.</Info>
      </Step>
      <Step title="Create a dataset">
        <CodeGroup>
          ```python Sync
          client.datasets().create("quickstart")
          ```
          ```python Async
          await client.datasets().create("quickstart")
          ```
        </CodeGroup>
      </Step>
      <Step title="Upload a file">
        Upload a file to your dataset by providing the document ID, the path to the file, and optional metadata:

        <CodeGroup>
          ```python Sync
          from pathlib import Path

          resp = client.dataset("my-docs").upsert_file(
              "report-2024",             # document ID
              Path("./report.pdf"),      # path to file
              {"title": "Annual Report"},  # optional metadata
          )
          ```
          ```python Async
          from pathlib import Path

          resp = await client.dataset("my-docs").upsert_file(
              "report-2024",             # document ID
              Path("./report.pdf"),      # path to file
              {"title": "Annual Report"},  # optional metadata
          )
          ```
        </CodeGroup>

        After the file has been uploaded, wait for it to be processed:

        <CodeGroup>
          ```python Sync
          client.dataset("my-docs").wait_for_handle(resp.handle)
          ```
          ```python Async
          await client.dataset("my-docs").wait_for_handle(resp.handle)
          ```
        </CodeGroup>
      </Step>
      <Step title="Ask a question">
        <CodeGroup>
          ```python Sync
          answer = client.ask("What are the key findings?", ["my-docs"])

          print(answer)
          ```
          ```python Async
          answer = await client.ask("What are the key findings?", ["my-docs"])

          print(answer)
          ```
        </CodeGroup>
      </Step>
    </Steps>
  </Tab>
  <Tab title="JavaScript SDK" icon="/icons/js.svg">
    <Steps>
      <Step title="Install JavaScript SDK">
        <CodeGroup>
          ```bash npm
          npm install topk-js
          ```
          ```bash yarn
          yarn add topk-js
          ```
          ```bash pnpm
          pnpm add topk-js
          ```
        </CodeGroup>
      </Step>
      <Step title="Initialize the client">
        ```typescript
        import { Client } from "topk-js";

        const client = new Client({
          apiKey: "your-api-key",
          region: "aws-us-east-1-elastica",
        });
        ```
        <Info>See [available regions](/regions) for a full list of supported regions.</Info>
      </Step>
      <Step title="Create a dataset">
        ```typescript
        await client.datasets().create("quickstart");
        ```
      </Step>
      <Step title="Upload a file">
        ```typescript
        const resp = await client.dataset("my-docs").upsertFile(
          "report-2024",    // document ID
          "./report.pdf",   // path to file
          { title: "Annual Report" },  // optional metadata
        );
        ```

        Wait for the file to be processed:

        ```typescript
        await client.dataset("my-docs").waitForHandle(resp.handle);
        ```
      </Step>
      <Step title="Ask a question">
        ```typescript
        const answer = await client.ask("What are the key findings?", ["my-docs"]);

        console.log(answer);
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
