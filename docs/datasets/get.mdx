---
title: "Get a dataset"
description: "Retrieve details about a specific dataset."
---

<Tabs>
  <Tab title="CLI" icon="terminal">
    ```bash
    topk dataset get --dataset my-docs
    ```
  </Tab>
  <Tab title="Python SDK" icon="/icons/python.svg">
    <CodeGroup>
      ```python Sync
      import os
      from topk_sdk import Client

      client = Client(
          api_key=os.environ["TOPK_API_KEY"],
          region=os.environ["TOPK_REGION"],
      )

      dataset = client.datasets().get("my-docs")

      print(dataset.name)
      ```
      ```python Async
      import os
      from topk_sdk import AsyncClient

      client = AsyncClient(
          api_key=os.environ["TOPK_API_KEY"],
          region=os.environ["TOPK_REGION"],
      )

      dataset = await client.datasets().get("my-docs")

      print(dataset.name)
      ```
    </CodeGroup>
  </Tab>
  <Tab title="JavaScript SDK" icon="/icons/js.svg">
    ```typescript
    import { Client } from "topk-js";

    const client = new Client({
      apiKey: process.env.TOPK_API_KEY!,
      region: process.env.TOPK_REGION!,
    });

    const dataset = await client.datasets().get("my-docs");

    console.log(dataset.name);
    ```
