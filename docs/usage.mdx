To monitor your usage and spend, go to the TopK [console](https://console.topk.io).

You can view your live [Usage](https://console.topk.io/projects/usage) and monitor your spend in [Billing](https://console.topk.io/billing).

## Accessing usage metrics

In the [TopK console](https://console.topk.io), select a project from the project list, then click **Usage** in the left sidebar. You can scope the view to:

- **Organization** — aggregate usage across all projects
- **Project** — usage for a single project and its datasets and collections
- **Dataset** / **Collection** — usage scoped to one dataset or collection

You can view the usage metrics up to the last 30 days.

## Billing and spend

Current spend and invoices are available on the [Billing page](https://console.topk.io/billing) in the console.

For pricing details, see the [TopK pricing page](https://topk.io/pricing).
