---

It offers a robust search that includes semantic search, vector search, keyword (BM25) search, sparse vector search, multi-vector search, hybrid search, reranking, and more.

Moreover, it's designed to support massive-scale datasets and is accessible via Python SDK or JavaScript SDK.
